Тег header задає «шапку» сайту або розділу веб-сторінки.
Тег nav створює навігацію між сторінками чи розділами сайту. Він допомагає користувачеві знайти потрібну інформацію і є важливим елементом для доступності.
Тег main використовується для позначення основної частини сайту.
Тег aside використовується для розміщення контенту, який не є основним, але пов'язаний із основним контентом на сторінці. Зазвичай, це бічна колонка з інформацією, наприклад, банери, навігаційні посилання, блоки соціальних мереж або реклама. Також у бічних колонках часто розміщують списки статей, що рекомендуються.
Тег section групує вміст сторінки, пов'язаний тематично або за змістом.
Тег article використовується для виділення вмісту, який є незалежним та самодостатнім.
Тег footer використовується для створення підвалу сайту. У цьому підвалі зазвичай містяться копірайт, контактна інформація, посилання на соціальні мережі, меню та інша допоміжна інформація.
